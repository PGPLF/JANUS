# JANUS - Validation du Modèle Cosmologique Bimétrique

Projet de validation du modèle JANUS par confrontation avec l'ensemble des observations cosmologiques disponibles.

## Objectif

Valider systématiquement le modèle cosmologique JANUS (modèle bimétrique avec matière et antimatière à masse négative) en le confrontant à toutes les observations disponibles et suivre en temps réel les nouvelles données observationnelles.

## Structure du projet

```
JANUS/
├── JANUS-PUB_REF/    # Publications de référence sur le modèle JANUS
├── JANUS-PUB_CRI/    # Publications et articles critiques du modèle
└── README.md
```

## Publications

Les résultats seront publiés régulièrement dans les grandes revues scientifiques académiques.

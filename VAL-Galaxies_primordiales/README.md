# VAL-Galaxies Primordiales

Ce dossier contient les travaux de validation du modèle JANUS par confrontation aux observations des galaxies primordiales.

## Objectif

Analyser les observations des galaxies primordiales, notamment celles du James Webb Space Telescope (JWST), et les comparer aux prédictions du modèle cosmologique JANUS.

## Observations Clés

- **Galaxies à très haut redshift** (z > 8, z > 10)
- **Masses stellaires inattendues** si tôt dans l'univers
- **Maturité structurelle** précoce des galaxies primordiales
- **Vitesse de formation stellaire** (SFR)

## Prédictions du Modèle JANUS

Le modèle JANUS prédit :
- Une formation plus rapide des premières structures
- Des galaxies plus massives et matures à haut redshift

## Structure

Les analyses seront organisées par :
- Observations JW[ST
- Comparaisons avec les modèles standard (ΛCDM)
- Prédictions du modèle JANUS
- Analyses statistiques

# JANUS-PUB_AUTRES

Articles de référence du modèle cosmologique standard (ΛCDM) et observations.

## Contenu

  - arXiv: astro-ph/9805201

- **Perlmutter_1999_SNe_Cosmology.pdf** - Perlmutter et al. (1999), ApJ 517, 565
  - "Measurements of Omega and Lambda from 42 High-Redshift Supernovae"
  - arXiv: astro-ph/9812133

### Paramètres cosmologiques
- **Planck_2018_VI_Cosmo_1807.06209.pdf** - Planck Collaboration (2020), A&A 641, A6
  - "Planck 2018 results. VI. Cosmological parameters"

  - arXiv: 1807.06209

  - "Cosmic Star-Formation History"

  - "Reionization after Planck"

- **Robertson_2015_Reionization_1502.02024.pdf** - Robertson et al. (2015), ApJ 802, L19

  - arXiv: 1502.02024

Ces articles servent de référence pour:
1. Comparaison avec le modèle JANUS
2. Contraintes observationnelles sur ΛCDM
3. Données haute-z (UV LF, réionisation)
4. Contexte historique de la cosmologie moderne

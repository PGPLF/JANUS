# JANUS_MODELE

Ce dossier contient une synthèse de toutes les équations scientifiques du modèle cosmologique JANUS.

## Objectif

Compiler et organiser les équations fondamentales du modèle bimétrique JANUS, y compris :

- Les équations de champ bimétriques
- L'équation de Schwarzschild gemellaire
- Les relations cosmologiques (H_0, Ω, évolution de l'univers)
- Les équations pour les observations (SNIa, BAO, CMB, etc.)

## Structure

Les équations seront organisées par thèmatique et accompagnées de leurs références bibliographiques.
